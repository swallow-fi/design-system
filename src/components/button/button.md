# Button

### 기본 상태

```html
<n-button>Button</n-button>
```

### disabled 상태

```html
<n-button disabled>Button</n-button>
```

### block 상태

```html
<n-button block>Button</n-button>
```

### icon 적용

```html
<n-button icon flat>
    <n-icon :name="'account_circle'"></n-icon>
</n-button>
<n-button>
    <n-icon :name="'account_circle'"></n-icon>
    account
</n-button>
```
